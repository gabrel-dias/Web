Parágrafos e quebras de linha

Para usar parágrafos, é só escrever entre as tags <p> e </p>.
O que "importa" para um parágrafo é apenas a tag, não o que estiver dentro dela.
Para que um parágrafo tenha uma quebra de linha, é preciso usar a tag <br> (break row).

Alguns símbolos especiais em HTML

Os comandos podem ser conferidos olhando o HTML da página
Todos são utilizados com "&" + "comando" + ";"
Exemplos de símbolos:
Registred Mark (®)
Copyright (©)
Trademark (™)
Seta para cima (↑)
Seta para baixo (↓)
Seta para esquerda (←)
Seta para direita (→)

Dinheiros:

Euro (€)
Libras (£)
Yenes (¥)
Centavos americanos (¢)
Delta maiúsculo (Δ)

Colocando emojis

Para colocar emojis basta colocar o código hexadecimal do emoji em questão com o prefixo "&#x" + <hexadecimal> + ";"
Exemplo: 🖖 💻
